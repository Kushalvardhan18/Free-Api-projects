# 🎯 Quotes Listings App

A simple and elegant web application for discovering, browsing, and sharing inspirational quotes.

🔗 Live Demo: https://quotes-listings-app.vercel.app/

---

## ✨ Features

- 📚 Browse a curated collection of quotes   
- 📱 Fully responsive design (mobile + desktop)  
- ⚡ Fast performance  
- 🌐 Deployed on Vercel  

---

## 🧠 Purpose

The Quotes Listings App is designed to provide quick access to motivational, inspirational, and thought-provoking quotes in a clean and distraction-free interface.

---

## 🛠️ Tech Stack

- **Frontend:** React / Next.js  / Bootstrap
- **Styling:** CSS / Tailwind CSS  
- **Deployment:** Vercel  

---

## 🚀 Getting Started (Local Setup)

```bash
# Clone the repository
git clone https://github.com/Kushalvardhan18/Free-Api-projects/tree/main/03_QuotesListingApp

# Navigate to project directory
cd 03_QuotesListingApp


# Install dependencies
npm install

# Start development server
npm run dev
