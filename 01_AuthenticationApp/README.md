# 🔐 Authentication App

A modern authentication web application built for secure user login, signup, and session management.

🔗 Live Demo: https://authentication-app-ashy.vercel.app/

---

## ✨ Features

- 🔑 User Signup & Login system  
- 🔒 Secure authentication flow   
- 🚪 Logout functionality  
- 📱 Fully responsive UI (mobile + desktop)  
- ⚡ Fast performance with Vercel deployment  


---



## 🛠️ Tech Stack

- **Frontend:** JavaScript,HTML  
- **Styling:** CSS / Tailwind CSS  
- **Deployment:** Vercel
