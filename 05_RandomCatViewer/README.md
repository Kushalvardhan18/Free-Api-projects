# 🐱 Random Cat Viewer

A simple and fun web app that displays random cat images with every refresh or click.

🔗 Live Demo: https://random-cat-viewer-eta.vercel.app/

---

## ✨ Features

- 🐾 Get a new random cat image instantly  
- 🔄 Refresh for endless cute cats  
- 📱 Fully responsive design (mobile + desktop)  
- ⚡ Fast loading via public Cat API  
- 🎨 Minimal and clean UI  

---

## 🧠 Purpose

This project is built for cat lovers who want a quick dose of cuteness. It demonstrates working with APIs and dynamic image rendering in a simple frontend app.

---

## 🛠️ Tech Stack

- **Frontend:** React js 
- **API:** https://api.freeapi.app/api/v1/public/cats/cat/random
- **Styling:** CSS / Tailwind CSS  
- **Deployment:** Vercel  

---

## 🚀 Getting Started (Local Setup)

```bash
# Clone the repository
git clone https://github.com/Kushalvardhan18/Free-Api-projects/tree/main/05_RandomCatViewer

# Move into project directory
cd 05_RandomCatViewer

# Install dependencies
npm install

# Start development server
npm run dev
