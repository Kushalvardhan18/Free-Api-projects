# 😂 Jokes UI Interface

A simple and fun **Jokes UI web application** built using ReactJs and tailwindCSS
This project fetches and displays random jokes from a public API in a clean and interactive interface.

---

## 🚀 Features

- 🎯 Fetch random jokes instantly
- 🎨 Clean and responsive UI
- 🌐 Uses public Joke API
- 📱 Mobile-friendly design

---

## 🛠️ Technologies Used

- reactJS
- TailwindCSS
- Public Joke API

---



## ⚙️ How to Run This Project

Clone the repository:
```bash
git clone https://github.com/Kushalvardhan18/Free-Api-projects.git
```
Navigate to the project folder:
```bash
cd Free-Api-projects/08_JokesUIInterface
```
Run following commands
```bash
npm install
npm run dev
```
